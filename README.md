# 🧸 Online Toy Store – E-commerce Website | BCA Project

![Project Banner](https://images.unsplash.com/photo-1601758123927-1965f17c1b9c)

## 📌 Project Overview

The **Online Toy Store** is a simple E-commerce website developed as part of a BCA practical project.  
This project demonstrates core web development concepts including shopping cart functionality, dummy checkout system, and SEO implementation.

The website is hosted using GitHub Pages.

## 🎯 Project Objectives

- Build a functional E-commerce website
- Implement Add-to-Cart feature using JavaScript
- Demonstrate On-Page SEO techniques
- Apply Off-Page SEO through GitHub optimization
- Host the website using GitHub Pages

---

## 🛍 Features

- 🛒 Add to Cart functionality  
- 💰 Dynamic total price calculation  
- 💳 Dummy payment system  
- 📱 Mobile responsive layout  
- 🔍 SEO-friendly HTML structure  

---

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- GitHub Pages (Hosting)

## 🔍 SEO Implementation

### On-Page SEO:
- Keyword-optimized title
- Meta description
- Proper heading hierarchy (H1–H3)
- Image ALT attributes
- Sitemap.xml
- Robots.txt

### Off-Page SEO:
- GitHub README optimization
- Keyword-rich repository name
- Repository description
- Topic tags
- Backlink sharing on social platforms
